# Space Invaders with Python
A simple space invaders game made using Python.

![image](https://user-images.githubusercontent.com/85440857/161965420-bf69f915-1a8d-4190-97c5-8289ff698217.png)

# Controls
| Key | Description |
| :---: | :---: |
| `Arrow Keys` | Move |
| `Space Bar` | Shoot Laser |
